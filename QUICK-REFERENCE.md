### Create a Branch
1. Click "Current Branch" dropdown (top center)
2. Click "New Branch"
3. Name your branch (use dashes, no spaces)
4. Click "Create Branch"

### Switch Branches
1. Click "Current Branch" dropdown
2. Select the branch you want
3. Your files will update automatically

### Pull Latest Changes
1. Click "Fetch origin" button (top right)
2. If there are updates, click "Pull origin"
3. Your local copy is now up to date

### Make and Commit Changes
1. Edit files in your code editor
2. Save your files
3. Go to GitHub Desktop
4. Review changes in "Changes" tab
5. Write a summary (required)
6. Write a description (optional but helpful)
7. Click "Commit to [branch-name]"

### Push Changes
1. After committing, click "Push origin" (top right)
2. Your changes are now on GitHub!

### Good Commit Messages
- **Be specific:** "Add user login form" not "updates"
- **Use action words:** "Fix", "Add", "Update", "Remove"
- **Keep it short:** Summary under 50 characters
- **Add details:** Use description for longer explanations

### Branch Names
- Use lowercase
- Use dashes instead of spaces: `my-feature` not `my feature`
- Be descriptive: `add-search-feature` not `feature1`
